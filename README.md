# Pasta House

Pasta House is a restaurant ordering website with a public menu, cart, Stripe checkout, order confirmation, order tracking, transactional emails, and an admin dashboard.

## Stack

- Frontend: React, Vite, TypeScript, Tailwind CSS
- Backend: Node.js, Express
- Database: PostgreSQL
- Payments: Stripe
- Emails: Resend

## Main features

- Public menu
- Cart and checkout
- Stripe payment flow
- Order confirmation
- Public order tracking
- Tracking recovery by email
- Admin authentication
- Admin dashboard
- Product, beverage, category, schedule, delivery and site settings management

## Security notes

Environment files are not committed. Configure local and production secrets through `.env` files on each environment.
